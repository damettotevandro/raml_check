Método GET (Listar)

/items:
  get:
    description: Retorna uma lista de itens.
    responses:
      200:
        body:
          application/json:
            type: array
            items: GeneratedType

Método POST (Avançado)

/items:
  post:
    description: Cria um novo item com validação e exemplos externos.
    body:
      application/json:
        type: !include types/ItemType.raml
        example: !include examples/ItemExample.json
    responses:
      201:
        description: Criado com sucesso.
        body:
          application/json:
            example: !include examples/ItemCreatedResponse.json
      400:
        description: Erro de validação dos dados.
        body:
          application/json:
            example: !include examples/ErrorResponse.json
